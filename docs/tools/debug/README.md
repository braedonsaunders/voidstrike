# Debug Tools

Utilities for debugging and testing game features.

## Tools

### effect-placer.html

A browser-based visual tool for positioning particle effects and visual elements in the game world.

**Usage:**
1. Open `effect-placer.html` in a browser
2. Load your scene or reference image
3. Click to place effect markers
4. Export coordinates for use in code

**Features:**
- Visual coordinate picking
- Grid snapping
- Coordinate export in multiple formats
